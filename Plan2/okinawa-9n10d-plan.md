# 오키나와 9박 10일 자유여행 일정 — Plan2 (재구성판)

## 개정 배경

Plan1을 동선·효율·밀도·아기 부담 기준으로 재검토하여 핵심 변경을 반영:

| 구분 | Plan1 | Plan2 |
|---|---|---|
| Day 3 나하 | 8곳 (슈리·킨조·시키나엔·太陽·나미노우에·쓰보야·코쿠사이·사카에마치) | **6곳 (시키나엔 제외, 동선 압축)** |
| Day 4 츄라우미 | 수족관만 | **수족관 + てつこのそば + 비세 후쿠기 + 에메랄드비치** (반경 2km 번들) |
| Day 6 (본 피크) | 후쿠기+에메랄드+てつこのそば+나키진성 (4곳 혼잡 명소) | **나키진성 + 오후 숙소 휴식** (경량 대응) |
| てつこのそば | Day 6 메인 | **Day 4 메인** (수족관 8분) |
| 해수욕 | 1회 (에메랄드) | **3~4회 (코우리, 에메랄드, 세소코, [오쿠마]) + 하트록** |
| 옵션 | — | **부세나 해중전망탑**(Day 5), **히지폭포 트레킹**(Day 7) |

|------|------|

| 08:30~11:30 | **츄라우미 수족관** — 쿠로시오 대수조, 돌고래 쇼(11:00), 터치풀 |
| 11:45~13:00 | **てつこのそば** — 모토부 고민가 소바집, 수족관에서 8분 (具志堅760) |

| 14:30~16:00 | **에메랄드비치** — 해양박공원 내, 얕은 물 수영, 아기 물놀이 |
| 16:30 | 숙소 복귀 (10km / 15분) |
| 저녁 | 숙소 자취 |

**플랜 B**:
- 아기 컨디션 불량: 수족관+てつこのそば만 보고 조기 귀환
- 비: 수족관(실내) + てつこのそば + 후르츠랜드(실내)로 교체
- てつこのそば 휴무 확인 필요 (출발 전 전화)

> 전날 편의점에서 수족관 할인 입장권 구매
> 이 번들은 **평일(5/1 금) + 오전 수족관 + 오후 해안 산책**으로 완전한 하루 구성. Day 6의 부담을 여기로 분산
